# 📚 Projeto Escolar - Sistema de Tarefas em React

Este projeto foi desenvolvido como parte de uma atividade escolar utilizando **React.js**. A aplicação consiste em um sistema simples de tarefas (To-Do List), permitindo adicionar, visualizar, marcar como concluída e remover tarefas. O objetivo é praticar os principais conceitos de desenvolvimento com React e aplicar boas práticas de programação front-end.

---

## 🛠️ Tecnologias Utilizadas

- ✅ [React](https://reactjs.org/) — biblioteca principal para criação da interface
- ✅ [JavaScript (ES6+)](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) — linguagem utilizada
- ✅ [Vite](https://vitejs.dev/) — ferramenta de build e servidor de desenvolvimento
- ✅ [Node.js](https://nodejs.org/) — runtime necessário para executar o projeto localmente
- ✅ [React Icons](https://react-icons.github.io/react-icons/) — biblioteca de ícones (opcional)
